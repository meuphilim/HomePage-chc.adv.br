Gaps e Margens

TokenUso
gap-6 lg:gap-8Grid de cards
gap-6 (flex-col)Flex interno de card — nunca usar mt-* nos filhos
mb-16Header para grid (seções completas)
mb-20Header para grid (produção acadêmica)
mb-10Header para galeria
mt-6Marquee container (espaço acima da galeria)
⚠️
Dentro de containers flex-col gap-*, nunca usar mt-* nos filhos diretos. O gap gerencia o espaçamento — margens manuais criam espaço duplo inconsistente.

Gradientes

⚠️
Usar sempre bg-gradient-to-* (Tailwind v3). Nunca bg-linear-to-* — sintaxe Tailwind v4, quebra silenciosamente todos os gradientes no projeto atual.
bg-gradient-to-b from-dark-900 via-dark-800 to-dark-900Fundo de seção
bg-gradient-to-r from-gold-600 to-transparentDivisor dourado
from-transparent via-white/10 to-transparentDivisor sutil
bg-gradient-to-br from-gold-600/10 to-transparentGlow de card hover
bg-gradient-to-t from-dark-900/80 to-transparentOverlay em imagem
bg-gradient-to-r from-dark-900 to-transparentMáscara lateral esquerda
bg-gradient-to-l from-dark-900 to-transparentMáscara lateral direita
