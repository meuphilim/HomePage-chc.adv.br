11 · Padrões

Motion & Animações

Todos os parâmetros de animação são controlados via ANIMATION constants. Nunca use valores hardcoded de duration ou delay.

GSAP · Constantes Visuais

DURATION.FAST: 0.3s
DURATION.NORMAL: 0.6s (Padrão)
DURATION.SLOW: 1.2s
STAGGER.FAST: 0.1s

Padrão canônico — useGSAPAnimation

Sempre prefira o hook padronizado em vez de gsap.context direto.

from: { y: 30, opacity: 0 }
to: { duration: ANIMATION.DURATION.NORMAL }

Framer Motion

Usado para molas de UI (tooltips, toggles).

type:"spring"
stiffness:280, damping:16
layoutId:"theme-option"
// Padrão de Animação de Entrada em Seções useGSAPAnimation(ref, { from: { y: 30, opacity: 0 }, to: { y: 0, opacity: 1, duration: ANIMATION.DURATION.NORMAL }, scrollTrigger: { start: ANIMATION.SCROLL_TRIGGER.START }, }); // Para listas com cascata (stagger) useGSAPAnimation(ref, { from: { y: 30, opacity: 0 }, to: { y: 0, opacity: 1, duration: ANIMATION.DURATION.FAST }, stagger: ANIMATION.STAGGER.SLOW, useChildren: true, });

14 · Diretrizes

Guia de Consistência

Regras de uso de ouro e anti-padrões para manter a coerência visual em novos componentes.

✓ Correto (Do's)
Usar font-mono tracking-widest uppercase em todos os badges de seção.

Usar ANIMATION.DURATION.NORMAL — nunca durations hardcoded.

Iniciar elementos animados com opacity: 0 via GSAP from, não via Tailwind opacity-0.

Cards com bg-dark-800 e border-border-subtle como padrão base.

Botões interativos: usar focus:ring-2 focus:ring-gold-500/50 obrigatoriamente para acessibilidade.

Texto de H2 sempre responsivo: text-4xl md:text-5xl.
✗ Evitar (Don'ts)
Usar font-medium em lugar de font-mono nos badges de seção.

Misturar text-3xl e text-4xl em H2s do mesmo nível hierárquico.

Usar bg-dark-800/80 ou bg-dark-900/30 em cards sem justificar o motivo arquitetural.

Aplicar classes do Tailwind como opacity-0 em elementos que dependem exclusivamente do GSAP para aparecer.

Criar Z-index soltos e fora da escala estabelecida (ex: z-[70]).

Botões CTA com rounded-lg em vez de rounded-full.

Checklist para Novos Componentes

✓ Layout

Usar SectionContainer como wrapper. py-20 md:py-32. Adicionar o § decorativo. Seção deve ter border-b border-white/5.

✓ Animação

Usar useGSAPAnimation com scrollTrigger. Respeitar as constantes de animação e evitar timeouts independentes.

✓ Cores

Fundo: dark-900. Cards: dark-800. Borda: white/5 → hover white/30 gold. Texto semântico: text-primary ou text-secondary.

✓ Tipografia

Títulos: font-serif. Labels/Badges: font-mono uppercase. Body: font-sans text-gray-300/400. Palavras-chave: gradient-gold.

✓ Imagens

Usar next/image com fill + sizes. Aplicar overlay gradiente escuro. Hover scale-105 com duration-700.

✓ Responsividade

Mobile-first obrigatório. Grades fluidas: grid-cols-1 → md:grid-cols-2 → lg:grid-cols-3.
